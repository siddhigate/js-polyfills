# js-polyfills
 Polyfills for JS methods:
 
 ## Array methods:
 This repo contains the polyfills of the following methods:
 ### Searching methods:
1. at
2. find
3. findIndex
4. findLast
5. findLastIndex
6. lastIndexOf
7. includes
8. indexOf
9. some
10. every

### Manipulation (Mutating) methods:
1. unshift
2. pop
3. push
4. shift
5. fill
6. reverse
7. sort
8. splice
9. copyWithin

### Iterative methods

1. forEach

### Map, filter, and reduce

1. map
2. filter
3. reduce
4. reduceRight

### Returning array iterator methods

1. keys
2. values
3. entries
 
### Other operations methods

1. slice
2. concat
3. flat
4. flatMap
5. join
